# NextJS Fundamentals

## Commands

### Manual Integration

1. `mkdir nextjs-fundamentals && cd nextjs-fundamentals`
2. `npm init -y`
3. `npm i next react react-dom`
4. `npm i -D typescript @types/react @types/node`
5. `npm i -D husky prettier pretty-quick`
6. `npm run dev`
7. `npm run build`
8. `npm run start`
